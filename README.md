# JOTERA
**CLI-Based Note-Taking Application using Python and SQL**

<img width="500" height="1000" alt="image" src="https://github.com/user-attachments/assets/d47d028d-8d4b-44ea-ba9e-c3e02d36ca92" />


A lightweight, command-line note-taking app inspired by Notion. Built for **CBSE Class XII Computer Science Project (2024-2025)**.

---

## 🚀 Features
- **Secure Authentication**: User registration & login with SHA-256 password hashing
- **Organized Notes**: Create pages + add/view/update/delete notes
- **Smart Search**: Keyword search within pages
- **Sort by Date**: View newest notes first
- **Timestamps**: Automatic creation/update tracking
- **SQLite Database**: Local, lightweight storage

## 🛠️ Tech Stack
- **Python** (Core language)
- **SQLite** (Database)
- **Libraries**: `sqlite3`, `hashlib`, `datetime`
